🌐 HTML5 Cheat Sheet

HTML5 | 2026 Updated

1기본 구조

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <meta name="viewport"
    content="width=device-width,
    initial-scale=1.0">
  <title>제목</title>
  <link rel="stylesheet"
    href="style.css">
</head>
<body>
  <!-- 콘텐츠 -->
  <script src="app.js"></script>
</body>
</html>

2Head 요소

태그설명
<title>페이지 제목 (탭에 표시)
<meta>메타데이터 (charset, viewport 등)
<link>외부 리소스 연결 (CSS 등)
<style>내부 CSS 스타일
<script>JavaScript 연결/삽입
<base>기본 URL 설정
<meta name="description"
  content="설명">
<meta name="keywords"
  content="HTML,CSS">
<meta property="og:title"
  content="OG제목">

3텍스트 & 제목

태그설명
<h1>~<h6>제목 (h1이 가장 큼)
<p>문단 (paragraph)
<br>줄바꿈
<hr>수평선
<strong> / <b>굵게 (의미/시각)
<em> / <i>강조/기울임
<mark>형광펜 강조
<del> / <ins>삭제선 / 밑줄
<sub> / <sup>아래첨자 / 위첨자
<blockquote>인용문
<code> / <pre>코드 / 서식유지
<span>인라인 그룹핑

4리스트

<!-- 순서 없는 리스트 -->
<ul>
  <li>항목 1</li>
  <li>항목 2</li>
</ul>

<!-- 순서 있는 리스트 -->
<ol type="1">
  <li>첫번째</li>
  <li>두번째</li>
</ol>

<!-- 정의 리스트 -->
<dl>
  <dt>용어</dt>
  <dd>정의</dd>
</dl>

5링크 & 이미지 & 미디어

<!-- 하이퍼링크 -->
<a href="url" target="_blank">
  링크텍스트</a>

<!-- 이미지 -->
<img src="img.jpg" alt="설명"
  width="300">

<!-- 비디오 -->
<video controls width="400">
  <source src="v.mp4"
    type="video/mp4">
</video>

<!-- 오디오 -->
<audio controls>
  <source src="a.mp3">
</audio>

<!-- iframe -->
<iframe src="url"
  width="600" height="400">
</iframe>

6테이블

<table>
  <thead>
    <tr>
      <th>이름</th>
      <th>점수</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>김</td>
      <td>90</td>
    </tr>
  </tbody>
</table>
속성설명
colspan="2"열 병합
rowspan="3"행 병합

7폼 (Form) — 입력 요소

<form action="/submit" method="POST">
  <label for="name">이름</label>
  <input type="text" id="name"
    placeholder="입력" required>
  <button type="submit">전송</button>
</form>
input type설명
text / password텍스트 / 비밀번호
email / tel / url이메일/전화/URL
number / range숫자/슬라이더
date / time날짜/시간 선택
checkbox / radio체크/라디오
file / color파일/색상 선택
hidden / submit숨김/제출

8폼 — 기타 입력

<!-- 드롭다운 -->
<select name="city">
  <option value="seoul">서울</option>
  <option value="busan">부산</option>
</select>

<!-- 텍스트영역 -->
<textarea rows="4" cols="50">
</textarea>

<!-- 필드셋 -->
<fieldset>
  <legend>개인정보</legend>
  ...
</fieldset>

<!-- datalist (자동완성) -->
<input list="fruits">
<datalist id="fruits">
  <option value="사과">
</datalist>

12Canvas & SVG & 기타
